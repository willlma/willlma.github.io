Projects

  • Monark is a proof of concept for a rich text editor without a user interface. It renders markdown in-place rather than showing separate plain and rich text panes.

  • Flowmodoro is a Mac menubar app that uses the motivation and burnout prevention aspects of the pomodoro technique without interrupting you if you've successfully entered a state of flow. I'm still testing it and will update this space when it's ready.

  • Marq is a Chrome extension that helps you go back to an article and keep reading where you left off. Just right-click a word to place a marq (a horizontal line across a page), and it'll be saved to your bookmarks. Next time you visit that page, the marq will reappear and the page will auto-scroll to its location. Install it or fork it.

  • Legible is a Firefox extension that toggles the visibility of fixed, scrolling, and overlay elements on entire domains for distraction-free reading.

  • The Firefox Extension Menu-Button is a module that allows you to use two-sided buttons (a main action button and a drop-down menu button) for your extensions.

  • Efficiency Timer is an Android app that can tell you, simply by keeping your phone in your peripheral vision, how efficiently you're doing a repetitive task.
